## 002-Meditation
*001-ChangeBackgroudColor*を改良した瞑想アプリケーションです。背景色の自然な変化を感じて、心を落ち着かせます。  
Low、Middle、Highの3種類で背景色の変化速度をかえることができます。  
  
[http://meditation-js.410gone.click/](http://meditation-js.410gone.click/)で、アプリケーションを公開しています。
